# second-brain

A personal aggregation hub for your **notes**, **links**, and **tasks**; your very own "second brain."

## Demo

[Demo Video](assets/demo.mkv)

## Features

- **Notes:** Quickly jot down ideas, thoughts, and reminders.
- **Links:** Save and organize links you want to revisit.
- **Tasks:** Track to-dos, set deadlines, and mark tasks as done.
- **Dashboard:** Get a quick overview of all your latest items and counts.
- **Authentication:** Secure login and registration with JWT.

## Tech Stack

- **Frontend:** React, TypeScript, Axios
- **Backend:** Node.js, Express
- **Database:** Prisma + PostgreSQL
- **Authentication:** JWT
- **Styling:** Custom CSS
